học tiếng anh theo cấu trúc cấp độ khác nhau.


sử dụng ngôn ngữ python, code tối ưu, code tự dộng hoá quy trình dữ liệu theo phân cấp, thứ tự.

ứng dụng hoạt động với python 3.13 trở lên

Tạo và kích hoạt môi trường ảo bằng venv hoặc uv , một gói Python dựa trên Rust nhanh chóng và trình quản lý dự án.

# venv
python -m venv .my-env

source .my-env/bin/activate


# uv
uv venv .my-env

source .my-env/bin/activate

cài đặt thư viện ứng dụng

pip install -r requirements.txt
